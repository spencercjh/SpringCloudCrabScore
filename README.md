#  基于Spring Boot2.x 的REST特色河蟹大赛评比管理系统

## 项目介绍
基于SpringBoot 2.x和Spring 5.x的SSM框架的Restful Android App服务端

Flask Python 服务端

## 软件架构

Spring Boot 服务端：REST (增POST 删DELETE 改PUT 查GET)

Python 服务端: 生成Excel文件
